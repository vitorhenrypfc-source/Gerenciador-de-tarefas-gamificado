#  Gerenciador de tarefas gamificado

##  Sobre o projeto

Este projeto foi desenvolvido como parte de uma atividade acadêmica da faculdade, com o objetivo de aplicar conceitos básicos de desenvolvimento front-end utilizando HTML, CSS e JavaScript.

## Ideia

O TaskMage é um gerenciador de tarefas com um sistema simples de gamificação. A proposta é transformar tarefas do dia a dia em pequenas conquistas, tornando a produtividade mais leve e motivadora.

## Funcionalidades

* Adicionar e remover tarefas
* Marcar tarefas como concluídas
* Sistema de experiência (XP)
* Progressão de níveis com dificuldade crescente

##  Como funciona

Cada tarefa concluída concede 10 pontos de XP.
Ao atingir o limite de XP, o usuário sobe de nível e o próximo nível exige mais experiência.

##  Tecnologias utilizadas

* HTML
* CSS
* JavaScript

##  Objetivo acadêmico

Praticar manipulação de DOM, lógica de programação e organização de código, além de desenvolver um projeto simples, funcional e com uma proposta criativa.
